# golang format 使用


=====%d,输出十进制====
110

=====%b,输出二进制====
1101110

=====%c,输出一个值的字符(char)====
a

=====%x,输出一个值的十六进制,每个字符串的字节用两个字符输出====
0xa
616263

=====%f,输出浮点型数值====
27.890000

=====%s,输出基本字符串====
I-am-batu

=====%q,输出带双引号的字符串====
“string”

=====%p,输出一个指针的值====
0xc82000a410

=====%v,输出结构体的对象值====
{1 2}

=====%+v,输出结构体的成员名称和值====
{x:1 y:2}

=====%#v,输出一个值的Go语法表示方式====
main.point{x:1, y:2}

=====%T,输出一个值的数据类型====
main.point

=====控制输出的宽度和精度====
| 12| 345|

=====输出宽度,同时指定浮点数====
| 1.20| 3.45|

=====输出宽度,同时指定浮点数====
|1.20 |3.45 |
